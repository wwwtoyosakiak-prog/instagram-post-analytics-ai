# サーバー起動手順

このアプリは、画面側のNext.jsサーバーと、必要に応じてPHPバックエンドを別々に起動します。

## 1. Next.jsだけで起動する場合

まずアプリのフォルダへ移動します。

```bash
cd "/Users/aokikousuke/Documents/Codex/2026-06-12/instagram-ai-instagram-api-instagram-csv/outputs/instagram-post-analytics-ai-full-20260614 2"
```

初回だけ、必要なライブラリを入れます。

```bash
npm install
```

開発用サーバーを起動します。

```bash
npm run dev
```

ブラウザで開きます。

```text
http://localhost:3000
```

## 2. 本番に近い形で起動する場合

cd "/Users/aokikousuke/Documents/Codex/2026-06-12/instagram-ai-instagram-api-instagram-csv/outputs/instagram-post-analytics-ai-full-20260614 2"

## 3. PHPバックエンドも使う場合

PHP連携を使う場合は、ターミナルを2つ開きます。

### ターミナル1: PHPサーバー

```bash
cd "/Users/aokikousuke/Documents/Codex/2026-06-12/instagram-ai-instagram-api-instagram-csv/outputs/instagram-post-analytics-ai-full-20260614 2/php-backend"
php -S 127.0.0.1:8080 router.php
```

### ターミナル2: Next.jsサーバー

```bash
cd "/Users/aokikousuke/Documents/Codex/2026-06-12/instagram-ai-instagram-api-instagram-csv/outputs/instagram-post-analytics-ai-full-20260614 2"
npm run dev
```

その後、ブラウザで `http://localhost:3000/settings` を開き、「PHP接続をテスト」を押してください。

## 4. OpenAI APIを使う場合

`.env.local.example` をコピーして `.env.local` を作ります。

```bash
cp .env.local.example .env.local
```

`.env.local` にAPIキーを入れます。

```env
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4.1-mini
```

設定後はNext.jsサーバーを一度止めて、もう一度起動してください。

## 5. サーバーを止める方法

起動中のターミナルで `control` + `C` を押します。

## 6. よくあるエラー

### `address already in use`

すでに同じポートでサーバーが起動しています。前のサーバーを `control` + `C` で止めるか、別のポートで起動してください。

```bash
npm run start -- --hostname 127.0.0.1 --port 3001
```

### `next: command not found`

`npm install` が完了していません。もう一度以下を実行してください。

```bash
npm install
```

### PHP接続に失敗する

PHPサーバーが起動しているか確認してください。

```bash

```
